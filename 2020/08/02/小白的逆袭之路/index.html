小白的逆袭之路

发表于 2020-08-02 更新于 2020-08-03
本文字数： 2.1k 阅读时长 ≈ 2 分钟

目前世界上已形成提高采收率四大技术系列，即化学法、气驱、热力和微生物采油，常用的气驱方式包括二氧化碳、氮气和烟道气等气体。CO2驱油是一种重要的提高采收率方法，在低渗透油藏、水驱废弃油藏等非常规油藏注气开发中效果显著。在CO2驱油过程中，大量的CO2注入地层，具有可观埋存碳潜力。碳减排是全球性的严峻挑战，我国碳排放量占世界碳排放量的29%，使用二氧化碳驱能够推进碳捕集与埋存。

1.二氧化碳驱油机理

根据注CO2的作用机理，分为CO2吞吐、非混相驱、混相驱。

1.1 CO2吞吐原理及操作流程

目前，CO2吞吐技术是一项应用较多的单井增油技术，操作流程如下：首先注入柴油，溶解有机质；然后注入缓蚀剂，形成保护膜，起防腐作用；再注入表面活性剂，驱使CO2 注入物性差、富含剩余油的油层，再注入乙二醇，以注入速度≤3t/h以内，注入压力控制在10MPa 以内注入液态CO2，焖井时间20 天，然后再依次注入乙二醇、缓蚀剂，最后注入顶替液，将井筒内的药剂顶入地层。
其技术原理主要有以下几点：
（1）降低原油粘度，改善流度比。CO2在油中有良好的溶解性，在温度一定的条件下，原油的粘度降低幅度随着压力增加、CO2溶解度增大而增大。CO2溶于水后使水碳酸化，水的粘度升高，油和水的流度趋向靠近，可改善油与水的流度比，从而扩大波及面积。
（2）原油体积膨胀。原油体积增大，促使充满油的孔隙体积也增大，水驱后留在油层中的不可动残余油随CO2溶解而膨胀，并被挤出孔道中，膨胀的油滴将水挤出孔隙空间，使孔隙压力升高，提高了地层原油的流动能力，形成了适合油流动的有利环境。
（3）降低油水界面张力。注入的CO2能够萃取和汽化原油中轻质烃，大量的轻烃与CO2 混合，可大幅度降低油水界面张力，从而改善油水流度比，扩大波及面积。
（4）提高地层渗透率。CO2溶解于水后略呈酸性，与地层基质相应地发生反应，从而酸解一部分杂质，在碳酸盐岩中能将部分岩石溶解，生成易溶于水的碳酸氢盐；在页岩中，由于PH 值的降低，碳酸稳定了粘土。另外，在CO2注入和返排过程中，在一定的压差下，一部分游离气对油层的堵塞物具有较强的冲刷作用，可有效地疏通因二次污染造成的地层堵塞。

1.2 CO2驱油原理及操作流程

根据IEA评估，全球适合采用CO2提高采收率技术开发的石油资源约有3×1086×108 bbl，相当于目前世界可采石油资源总量的14%28%。其中，CO2混相驱提高原油采收率效果明显高于CO2非混相驱。2014年，美国CO2驱的项目总数为137个，提高采收率的总产油量为1371×104 t/a，占美国EOR总产油量的43.1%，提高采收率幅度在7%~20%之间。其中，混相驱项目数达到117个，产油量为1264×104 t/a，占总产油量的92.20%；非混相驱项目个数为8个，产油量为107×104 t/a，占总产油量的7.80%。
混相驱的基本机理是驱替剂(注入的混相气体)和被驱剂(地层原油)在油藏条件下形成混相，消除界面。从混相原理上可知，随着压力增加，即使是用贫气驱动含中间分子量烃较少的重油，它们之间也有可能达到混相，但要求的混相压力极高，这在油藏注气工程中有时是不可能达到的，此时注气只能是非混相驱替。为了达到较高的驱油效果，CO2提高采收率技术以混相驱为主。对不能实施CO2混相驱开发的油藏，要么考虑其他驱油方式开发，要么采取一定的措施或技术手段，促使CO2非混相驱实现混相驱或近混相驱，从而达到最大幅度提高CO2驱整体技术经济效果的目的。我国大部分油藏属于陆相沉积，CO2驱的最小混相压力(Minimum Miscibility Pressure，MMP)较高，大部分油藏在地层压力下不能实现混相驱，阻碍了CO2-EOR 技术的大规模推广应用。我国18个油藏区块注CO2的最小混相压力见表1

2. N2驱油机理

（1）保持地层压力
在对一些特殊的原油进行开采时所具备的特殊条件是保持一定的压力, 将压力值维持在一定的数值内。注氮气方法能将压力维持在相应的数值之内可以很好的解决残余油量较多采收率不高等问题, 从而提高采收率。
（2）易混相气体段塞混相驱
注氮气推动易混相气体段塞混相驱的方法能解决了氮气驱油在运用上所出现的技术难题并且良好地利用氮气的价格低, 来源广等优势。此做法不仅能够利用二氧化碳和烃类气体易混相的特点, 相比之下, 氮气在成本方面优势更加明显。段塞尺寸如果能够选择适当, 在氮气推动混相段塞时, 会比持续注人氮气效果更好, 可以有效的提高经济效益。
（3）氮气连续注入混相驱
氮气驱油时, 气体与原油抽取的次数增加和碰触作用过程中, 必然会导致压力值不断变化, 并且出现上升趋势。油中的中间烃大量进人到气体中, 在某一临界点时, 原油和氮气就会形成特殊的现象。为了让氮气与许多油藏原油达到蒸发气汽驱动态混, 相完全可利用氮气难与原油发生接触混相的原理。
（4）氮气和水交替注入混相驱机理
采用氮气和水交替注人混相驱机理是由于原油粘度比氮气高，因而能有效提高采收率。可以改善如降低气体流窜等在驱油时产生的一系列影响。聚合物嵘乏面活性剂二元体系驱油方法能使聚合物钻弹性得到充分发挥，乳状液带来的不良影响可以得到控制, 可以完全清除碱在地层及井引起的结垢现象。
